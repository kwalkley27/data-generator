# Examples

This directory contains sample outputs generated by the data-generator app.

## Customer Data Sample

**Input Schema:**
- `name`: realistic full names
- `email`: professional email addresses
- `age`: ages between 25-65
- `city`: major US cities
- `occupation`: common job titles

**Command:** Generated 5 records

**Output:** See [sample_customer_data.json](sample_customer_data.json)

---

## Product Catalog Sample

**Input Schema:**
- `product_name`: creative product names for a tech store
- `price`: prices between $10-$500
- `category`: electronics categories
- `in_stock`: boolean values

**Command:** Generated 5 records

**Output:** See [sample_product_catalog.json](sample_product_catalog.json)

---

## How to Use These Examples

1. Open the Streamlit app
2. Define similar field schemas
3. Click "Submit" to generate your own data
4. Click "Download" to save the results
