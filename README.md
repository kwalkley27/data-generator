# data-generator

An app that allows users to generate sample data by defining fields with natural language.

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/data-generator.git
   ```
2. Install the dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Set the `GEMINI_API_KEY` environment variable with your Google API key.

## Usage

1. Run the Streamlit app:
   ```bash
   streamlit run src/app.py
   ```
2. Open the app in your browser.
3. Define the fields and their value descriptions.
4. Enter the number of records to generate.
5. Click "Submit" to generate the data.
6. Click "Download" to save the data to a file.
